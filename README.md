## 🤸 Quick Start

Follow these steps to set up the project locally on your machine..

**Prerequisites**

- [Git](https://git-scm.com/)
- [Node.js](https://nodejs.org/en)
- [npm](https://www.npmjs.com/)
- [eas-cli](https://www.npmjs.com/package/eas-cli)
- [expo-cli](https://www.npmjs.com/package/expo-cli)
- [Setup Expo IOS Simulator](https://docs.expo.dev/workflow/ios-simulator/)
- [Setup Expo Android Studio Simulator](https://docs.expo.dev/workflow/android-studio-emulator/)
- [Login to Expo](http://expo.dev/)

**Cloning the Repository**

```bash
git clone https://github.com/YenkoDev/tales-of-siren-mobile.git
cd tales-of-siren-mobile
```

**Installation**

```bash
npm install
```

**Running the Project**

```bash
npm run prebuild-clean
npm run android
npm run ios
```

**Check Dependecy Error**

```bash
npm run check-install
```

**NPM Check Updates**

```bash
npm run check-updates
```
